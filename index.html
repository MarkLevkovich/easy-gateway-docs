overview

A simple to use and lightweight API Gateway for microservice architecture.

✨ Features

- ✅ Easy setup via YAML
- ✅ CLI interface
- ✅ Middleware system
- ✅ Routing with prefixes
- ✅ Rate limiting
- ✅ Logging

Next will be authentication, caching, and etc!

install

$ pip install easy-gateway

Requirements: Python ≥ 3.7, no dependencies

usage

Create a config.yaml based on the template in the examples (in the root directory)

add server settings:

server:
    host: "0.0.0.0"
    port: 8000

Next:

add routes:

routes:
  - path: "/bin/*"
    target: "https://httpbin.org/"
    description: "Echo Server"
>>> FOR EXAMPLE: user/* -> for a url with any prefix after user.
but user/ -> for a clear url

Select middlewares:

now only logging and rate-limit

middlewares:
  - name: "LoggingMiddleware"
    enabled: true

  - name: "RateLimitMiddleware"
    enabled: true
    requests_per_minute: 5 # select your own limit

Setup CORS:

add you origins:


    cors:
      allow_origins:
        - "myfront.com"
        - "testreact.space"

To start gateway:

run it in the root directory

>>> easy-gateway -c PATH-TO-YOUR-CONFIG
    # or --config or just easy-gateway if your config is in root directory

example of config.yaml

server:
  host: "0.0.0.0"
  port: 8000

routes:
  - path: "/users/*"
    target: "https://mysite/list/users"
    description: "Echo Server"

  - path: "/admin"
    target: "https://mysite/admin"
    description: "Echo Server"

  - path: "/chat"
    target: "https://mysite/chat"
    description: "Echo Server"

middlewares:
  - name: "LoggingMiddleware"
    enabled: true

  - name: "RateLimitMiddleware"
    enabled: true
    requests_per_minute: 5
